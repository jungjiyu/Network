MAC Media Access Control 주소
	: 네트워크 연결되는 장비들이 가니는 48비트(=6Octet) 의 "고유한" 주소
		:Octet==8비트 한마디
	: 물리적 주소
	: 구성
		앞의 24비트: OUI Organizational Unique Identifier
			: 어느 회사에서 생산했는지 나타냄
			: IEEE 에서 발급해준다
				: Institute of Electrical and Electronics Engineers
				: I 쓰리플E
				: 전기전자 기술 학회 
					: 세계에서 가장 큰 학회

				: 빛통신
					: 선생님이 봤을땐 대중화에 실패할 것 같다고 한다. 빛 간섭 그런것 때문에

OSI 7 Layer 모델
	: 통신이 일어나는 과정 7단계
	: 국제 표준화 기구(ISO)가 1984년에 발표
	: 왜 굳이 나눴나>> (전체적으로 보수하는게 아니라)단계별로 소프트웨러를 보수 가능+흐름파악용이

	: 계층
		1계층 - Physical Layer 물리계층
			:  전기적, 기계적, 기능적인 특성을 이용해서 통신 케이블로 데이터를 전송하는 계층
			: 비트 단위로 통신(= 0 , 1 로 나타나진다 = 전기적으로 on off 다)
			: 단지 데이터를 전달만 하고 받으려는 데이터에 대한 에러에 대한 검사는 안함
			: 대표 장비) 리피터, 허브


		2계층 - Data Link Layer 데이터 링크 계층
			: 신뢰성있는 전송을 보장하기 위한 계층
				: 물리 계층을 통해 송수신되는 데이터에 대하 오류를 검사함
			: 프레임 단위로 MAC 주소를 부여하여 통신
				: 프레임) 2 계층에서 데이터를 전달하는 단위
			: 브리지, 스위치 를 통해 맥 주소를 가지고 물리계층에서 받은 정보를 전달함

		3계층-네트워크 계층 Network Layer
